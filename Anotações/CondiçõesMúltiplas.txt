Condição Múltipla:

switch(expressão){
	case valor1:
		bloco
		break

	case valor2:
		bloco
		break

	case valor3:
		bloco
		break

	default:
		bloco
		break
}

Cada "case" vai o valor esperado para realizar aquele bloco, esse valor
é fixo, ou seja, ao ler a "expressão" o programa vai rodar direto
o "case" que tenha o mesmo "valor", depois disso o programa
bate no "break" que é onde ele para de excutar o programa,
cada bloco precisa ter um "break" obrigatoriamente se não tiver um "break"
após o comando e próximo bloco também será executado, o uso de "default" é opcional
funcionando com um "else" que também precisa ter um "break"

var agora = new Date()
var diaSem = agora.getDay() **Nesse exemplo o dia é quarta-feira porém o JS
				identifica como números, como o exemplo abaixo.**
/*
   0 = Domingo
   1 = Segunda
   2 = Terça
   3 = Quarta
   4 = Quinta
   5 = Sexta
   6 = Sábado    
*/

switch (diaSem) {
    case 0:
        console.log('Domingo')
        break
    case 1:
        console.log('Segunda')
        break
    case 2:
        console.log('Terça')
        break
    case 3:
        console.log('Quarta')  **No caso seria esse bloco a ser executado**
        break
    case 4:
        console.log('Quinta')
        break
    case 5:
        console.log('Sexta')
        break
    case 6:
        console.log('Sábado')
        break
    default:
        console.log('[ERRO] Dia inválido!')
        break
}

Uma limitação do switch é que ele só funciona com números inteiros
e strings, o que o torna mais limitado que os 'IFs', porém 
facilita bastante em situações pontuais.
