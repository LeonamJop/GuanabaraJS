Variável simples: consegue armazenar um valor por vez.

Variavel composta: é capaz de armazenar vários valores em uma mesma estrutura.

Para cada valor você teria que criar um var/let 
EX: var vagaAuto1 = carro1 , vagaAuto2 = carro2, vagaAuto3 = carro3...

Agora imagina se fossem 200 vagas de automovel?
Ter que criar 200 "var/let" é complicado,
melhor usar as variaveis compostas,

EX: var vagaA = [carro1,carro2,carro3]
Serão colocados em suas repequitivas vagas:
carro1 na vaga 0, carro2 na vaga 1, carro3 na vaga 2.


Esse variavel "vagaA" é do tipo array/vetor.

VagaA = array/vetor
Espaço de armazenamento = elemento
carro1 = Conteúdo/valor
vaga 0 o "0" é o índice/chave

Exemplo em código pra ficar mais fácil o entendimento:

var/let num = [5,8,4]

Ou seja
num
índice 0 = valor 5, índice 1 = valor 8, índice 2 = valor 4

Acresentando valor: num[3]=6
Ou seja
num
índice 0 = valor 5, índice 1 = valor 8, índice 2 = valor 4, índice 3 = valor 6

Outro forma seria:
num.push(9)
ficaria como
num
índice 0 = valor 5, índice 1 = valor 8, índice 2 = valor 4,
índice 3 = valor 6, índice 4 = valor 9

Para saber o comprimento do seu vetor utilize .length
EX: num.length, resultado de será 5 pois tem 5 elementos

O .sort() deixa todos os valores em ordem crescente 

EX: índice 0 = valor 4, índice 1 = valor 5, índice 2 = valor 6,
índice 3 = valor 8, índice 4 = valor 9

Usando lógica para apresentar os valores:

for(let pos=0; pos < num.length; pos++){
	console.log(num[pos])
}

simplificando:

for(let pos in num){
	console.log(num[pos])
}

Buscando valores dentro do vetor:
num.indexOf(valor)

num.indexOf(8) vai retornar a posição que nesse caso é a 3.

Se você colocar um valor que não existe será retornado o índice -1
que significa que ele pesquisou mas não encontrou o valor.
