Condições Aninhadas: consiste e uma condição composta com outras condiões dentro dela.

if(condição1{
	bloco1
}else {
	if(condição2){
		bloco2
	}else{
		bloco3
	}
}

Se a 1° condição for falsa ele testara a 2° condição e excutara o 2° bloco, caso a 2° condição seja verdadeira,
se a 2° condição também for falsa executara o 3° bloco. Você pode por mais condições caso aja necessidade,
adicionando uma 3°, 4° condição e assim por diante.

Usando "else if":
var idade = 22
if(idade < 16){
	console.log('Não vota')
}else{
	if(idade >=16 && idade < 18){
		cosole.log('Voto opcional')
	}
}

No segundo "if" a condição "idade >= 16" é redundante,
também não existe condisão para "idade > 18",
também não a condição para "idade > 65", 
segue um exemplo de como melho fazer esse teste:

var idade = 22
if(idade < 16){
	console.log('Não vota')

}else if(idade < 18 || idade > 65){	
	cosole.log('Voto opcional')	

}else{
	console.log('Voto obrigatório')

}
