Variaveis são usadas para armazenar e organizar dados.
Sendo numeros, nomes ou booleano
ou seja number, string ou boolean

var vaga1 = carro1
no caso do exemplo de cima a "vaga1" recebe "carro1" caso for usar a "vaga1" novamente a informação
do "carro1" será substituida pela infomação nova.

o nome de variavel é chamado de identificadores.

Os identificadores só podem começar com 'LETRA', '$' OU '_' EX: var carro, var $dinheiro ou _id;

Não podem começar com números EX: var 1carro;

É possível usar 'letras' ou 'números' EX: var carro1;

É possível usar 'acentos' e 'símbolos' EX: var média;

Não podem conter espaço Ex: var id person: nesse caso o "espaço" pode ser substituido por '_' ex: var id_person;

Não pode usar palavras reservadas Ex: var alert, var var.
