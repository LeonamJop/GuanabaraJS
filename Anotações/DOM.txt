DOM: Document Object Model (Modelo de Objetos para documentos). 
É básicamente um conjunto de objetos no seu navegador que vai dar acesso 
aos componentes internos do seu website.
Não funiciona no nodeJS. Está presente quando está rodando JS dentro do seu navegador.

Árvore DOM:

window
{ location (url do seu site),
	document (documento atual){
		html{head (parte de cabeçalho)
		{meta, title.},
		body (parte de corpo)
		{h1, p, p{strong}, div}
		}
	},
	history (histórico do seu site)
}Representação do exercicio ex005
Cada um acima são conhecidos como "elementos" 
Podendo ser selecionados (5 principais)
por Marca (getElementsByTagName())
por ID (getElementById())
por Nome (getElementsByName())
por Classe (getElementsByClassName())
por Seletor querySelector() ou querySelectorAll()

Toda <div> é representada por uma '#'
Toda class é representada por um '.'
